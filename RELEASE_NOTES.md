# PulseClick v1.0.0 发布说明

## 发布状态

PulseClick v1.0.0 是面向 Windows 10/11 的首个稳定便携版本。用户下载后无需安装，解压并双击 `PulseClick.exe` 即可运行；删除整个软件目录即可卸载。

## 主要功能

- 支持左键、右键、中键自动点击。
- 支持单击、双击、三击。
- 点击间隔支持 1ms 到 3600000ms。
- 支持无限循环和指定次数。
- 支持跟随光标和固定坐标。
- 支持屏幕覆盖层拾取固定坐标。
- 支持默认 F6 全局启停热键和 Ctrl/Alt/Shift/Win 组合键。
- 支持系统托盘显示主窗口、开始、停止和退出。
- 支持配置持久化、配置档案、日志文件和随机延迟。
- 支持实时运行状态、CPS、点击计数和运行时长显示。

## 便携数据

首次运行后，程序会在 `PulseClick.exe` 同级目录自动创建运行时数据：

```text
data\config.json
data\profiles\
data\logs\pulseclick.log
```

配置、档案和日志不会写入 Windows 用户目录、注册表或系统目录。

## 分发文件

正式分发时只需要提供：

```text
PulseClick.exe
```

不要把测试运行生成的 `data\` 一起打包给用户。

## 注意事项

- 本版本仅支持 Windows 10/11。
- 鼠标点击使用 Windows `SendInput`，全局热键使用 Windows `RegisterHotKey`。
- 自动点击工具可能被部分游戏、反作弊或安全软件限制。
- 用户应遵守相关平台规则，并只在合法、合规、被允许的场景使用自动点击功能。
